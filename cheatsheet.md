# Keymap cheatsheet

> [!Note] <leader> for this repo is the space bar


|*Feature*|*Kepmap*|
|---------|--------|
|Fuzzy find files|Ctrl + p|
|Ripgrep|Ctrl + f|
|Split vertical|Ctrl + l|
|Split horizontal|Ctrl + b|
|Close split|Ctrl + q|
|Flash find|s + chars|
|Close buffer|Ctrl + w|
|Indent block|Shift + >|
|Unindent block|Shift + <|
|Jump endline|$|
|Jump start line|0|
|Docs preview|Shift + k|
|Toggle comment|g + c + c|
|Block comment|highlight + g + c|
|Toggle diagnostics (all files)|<leader> + t + d|
|Toggle diagnostics (buffer)|<leader> + t + b|
|Show all todos|<leader> + t + t|
|Search todos|<leader> + f + t|
|Jump to top|Ctrl + Home|
|Jump to end|Ctrl + End|
|Toggle terminal|Ctrl + t|
|Dismiss notif|<leader> + n + d|
|Jump to dashboard|<leader> + h|
|Markdown preview|<leader> + m + p|
